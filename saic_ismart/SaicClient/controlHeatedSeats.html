Controls the heated seats on vin.

Pass driverLevel and/or passengerLevel to set each seat independently. Both default to HeatLevel.off.

Before calling, check VehicleFeatures.heatedSeatCapability to verify the vehicle supports heated seats and to determine which HeatLevel values are valid.
