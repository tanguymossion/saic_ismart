VehicleFeatures

VehicleFeatures class

Feature-detection wrapper around a Vehicle.

All accessors return false / HeatedSeatCapability.none when the relevant configuration item is absent — i.e. unknown is treated as unsupported for safety.

final features = vehicle.features;
if (features.hasSunroof) { ... }
