BasicVehicleStatus

BasicVehicleStatus class

Core vehicle telemetry snapshot.

All fields are int?. The API omits fields that do not apply to the vehicle's powertrain:

  • BEV: fuelRange/fuelLevelPrc are null; fuelRangeElec/vehElecRngDsp are present.
  • ICE: electric range fields are null; fuel fields are present.
  • PHEV: both sets of fields may be present.

Source: api/vehicle/schema.py:BasicVehicleStatus
